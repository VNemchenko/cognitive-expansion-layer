# Contributing

This repository is primarily a defensive publication and concept documentation repository.

## Contributions

Contributions may be accepted for:

- typo fixes;
- clarification of terminology;
- references to relevant prior art;
- documentation improvements;
- issue reports about ambiguity.

Contributions are not expected for:

- production Brownyx code;
- private prompt design;
- scoring heuristics;
- model routing;
- infrastructure implementation;
- runtime data;
- claims about consciousness.

## Contribution Licensing

By contributing documentation to this repository, you agree that your contribution may be distributed under the repository documentation license.

## Discussion Style

Please keep discussion:

- precise;
- non-mystical;
- technically grounded;
- clear about speculation;
- respectful of the boundary between architecture research and consciousness claims.
