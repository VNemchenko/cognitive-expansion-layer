# Security Policy

This repository is documentation-only and does not contain production [Brownyx](https://brownyx.com) source code.

## Do Not Submit

Please do not submit:

- API keys;
- credentials;
- private prompts;
- private runtime traces;
- private memory dumps;
- production logs;
- infrastructure configuration;
- customer data;
- proprietary implementation details.

## Reporting

If you notice accidental exposure of sensitive information, open a private communication channel with the repository owner rather than creating a public issue.

## Scope

This policy applies only to this documentation repository.

Production [Brownyx](https://brownyx.com) security issues should be reported through the private [Brownyx](https://brownyx.com) project channels.
